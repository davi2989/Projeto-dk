Estrutura de Dados e Modelo de Banco

Esta seção apresenta o modelo simplificado de armazenamento de dados utilizado para simular o funcionamento do sistema de recomendação da Netflix dentro do escopo acadêmico do projeto.

O Diagrama Entidade-Relacionamento (DER) demonstra como as informações fundamentais do sistema são organizadas e conectadas. Embora a infraestrutura real da Netflix seja muito mais complexa, este modelo foi desenvolvido de forma simplificada, focando apenas nos elementos essenciais para a personalização de recomendações.

As principais entidades do modelo são:

1. Usuário: armazena informações básicas sobre cada perfil, como identificação e preferências gerais.

2. Conteúdo: representa os filmes, séries e demais títulos disponíveis no catálogo da plataforma.

3. Histórico: registra as interações do usuário com os conteúdos, incluindo datas, horários e tempo assistido.

4. Sugestão: conecta usuários a conteúdos sugeridos com base em seus padrões de consumo e comportamento.

Essa estrutura permite que o sistema identifique padrões, relacione preferências e gere sugestões personalizadas de maneira lógica e eficiente.
