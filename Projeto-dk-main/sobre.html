Sobre o Projeto

Este dossiê digital foi desenvolvido com o objetivo de investigar, modelar e apresentar de forma visual o funcionamento dos sistemas de recomendação utilizados pela Netflix, unindo tecnologia, design e análise de dados.

Davi Kleemann — Responsável pela modelagem de dados, estrutura lógica do sistema e organização conceitual do projeto.

Kauã Sander — Responsável pelo design visual, estrutura web e experiência do usuário.

O projeto foi desenvolvido com foco educacional, buscando traduzir conceitos complexos de inteligência artificial em uma experiência acessível, didática e visualmente envolvente.
