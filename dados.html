Os Dados — Modelo de Banco

As tabelas principais são: Usuário, Título, Histórico, Recomendação e Gênero.

O Diagrama Entidade-Relacionamento (DER) representa como os dados principais do sistema de recomendação são organizados e conectados. Embora a Netflix real utilize uma estrutura muito mais avançada, este modelo foi simplificado para atender às necessidades do trabalho, focando apenas na base mínima necessária para gerar recomendações personalizadas.

O DER mostra como as informações são distribuídas e como cada entidade se relaciona com as demais. No modelo criado, trabalhamos com cinco entidades principais:

1. Título:

Representa os filmes, séries e produções disponíveis no catálogo da plataforma. Cada título pertence a um gênero e pode aparecer no histórico de vários usuários.

2. Gênero:

Indica a categoria de cada título, como ação, drama, comédia, romance, entre outros. Um único gênero pode estar associado a diversos títulos do catálogo.

3. Usuário:

Representa cada pessoa que utiliza a plataforma. O usuário possui um histórico de títulos assistidos, além de receber recomendações personalizadas.

4. Histórico:

Registra todos os títulos que o usuário já assistiu. Ele conecta o usuário aos conteúdos consumidos, permitindo que o sistema entenda suas preferências.

5. Recomendação:

Mostra quais títulos foram sugeridos aos usuários com base no perfil, comportamento e padrões identificados. Cada recomendação liga um usuário a um título que o sistema considera relevante para ele.
